HTML: Hypertext Markup Language

HTML is the standard markup language used to create the structure of web pages. It defines elements such as headings, paragraphs, images, links, and more, giving the browser instructions on how to display content.

CSS: Cascading Style Sheets

CSS is used to style and format the HTML content. It controls the visual appearance of web pages, including layout, colors, fonts, spacing, and responsiveness across different devices.

JS: JavaScript

JavaScript is a programming language that adds interactivity and dynamic behavior to web pages. It allows developers to create animations, handle user input, update content without reloading the page, and much more.
